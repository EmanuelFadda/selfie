# Progetto selfie 

Questo è il progetto del corso di Tecnologie Web della laurea di Informatica dell'Università di Bologna :

La repository comprende :
- client  (vue.js)
- backend  (express.js)


# How to:

## Runnare il client

    cd client
    npm run serve
    

## Runnare il backend

    cd backend 
    node main.js
